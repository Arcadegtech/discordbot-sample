#in config.json
1. Please enter client ID & token of bot.
2. Enter logs channel id in console ID
3. Enter your guild id & choose a prefix.

#installing modules
1.run command "npm i" in main directory to install modules.

#running the bot
1. Rename "run.config.js.txt" to "run.config.js"
2. then run the bot with PM2 from "run.config.js"
3. Its preferred to use node js 17 +
